## 一套简单的 webapck 配置文件 (基于webpack2)
能够满足基础使用

* 支持es6 
* 支持热重载
* less语法


## 目录结构

![](http://oo88qkz8q.bkt.clouddn.com/17-7-14/14580594.jpg)


### 说明: 

`config`为放置配置文件目录:

* build-config为构建配置
* dev-config和dev-server为开发配置

`src`为加载资源目录:

* `main.js`为项目入口文件
* js/css/img 分别为对应的资源

## dev 命令:  
 
 npm run dev 


## build 命令:
 
 npm run build
